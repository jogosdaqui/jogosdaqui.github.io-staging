    Promoção sobre Vortex Game Studios
    Post escrito por Diego Giacomelli
    Colabore você também

    A Vortex Game Studios nos enviou 9 chaves do seu novo game Space Lander para sortearmos entre nossos leitores.

    Para participar da promoção, siga os passos abaixo:

    • Os prêmios deste sorteio serão 9 chaves para o game Space Lander da Vortex Game Studios. Os ganhadores serão os 9 primeiros colocados na lista gerada aleatoriamente pelo random.org;

    • Para concorrer, siga o @jogosdaqui, deixe um comentário neste post e poste uma das frases abaixo no Twitter:
      • Quero concorrer a 9 chaves para o game Space Lander da Vortex Game Studios no @jogosdaqui: https://jogosdaqui.github.io/jogosdaqui.github.io-staging/2015/05/13/promocao-do-game-space-lander
      • Conheça o game brasileiro Space Lander da Vortex Game Studios no @jogosdaqui: https://jogosdaqui.github.io/jogosdaqui.github.io-staging/2015/05/13/promocao-do-game-space-lander
      • Isaac Newton + David Bowie + Gameboy + Space Lander no @jogosdaqui - https://jogosdaqui.github.io/jogosdaqui.github.io-staging/2015/05/13/promocao-do-game-space-lander
      • Um pequeno pixel para um homem, um pixel gigantesco para a humanidade - Space Lander no @jogosdaqui - https://jogosdaqui.github.io/jogosdaqui.github.io-staging/2015/05/13/space-lander
    • Concorrerão todas as pessoas que seguirem todos os procedimentos descritos acima, até as 18:00 horas (horário de Brasília) do dia 15/05/2015 (sexta-feira).

    Bom game e boa sorte a todos!
